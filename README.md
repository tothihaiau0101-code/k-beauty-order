# 🇰🇷 K-Beauty & K-Pop Order
> Shop mỹ phẩm & album Hàn Quốc chính hãng

## Stack
- Frontend: HTML/CSS/JS (Vanilla, dark theme)
- Backend: Python 3 (stdlib only)
- Storage: JSON files

## Cài đặt
```bash
cd tools && cp .env.example .env && python3 telegram_bot.py
```

## Tính năng
- 📦 Catalog sản phẩm
- 📩 Form đặt hàng tương tác
- ⚙️ Admin dashboard (đơn hàng + tồn kho + doanh thu)
- 🔔 Telegram bot thông báo
- 📊 Xuất nhập tồn real-time

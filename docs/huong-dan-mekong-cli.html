🏗 Kiến Trúc 6 Tầng

Mekong CLI là nền tảng kinh doanh vận hành bằng AI, tổ chức theo 6 tầng từ chiến lược đến thực thi.

1STUDIOSáng tạo — brainstorm, design, branding, content
2FOUNDERChiến lược — fundraise, intel, binh-phap, SWOT
3BUSINESSKinh doanh — sales, marketing, finance, CRM
4PRODUCTSản phẩm — plan, sprint, roadmap, launch, ship
5ENGINEERINGKỹ thuật — code, fix, review, deploy, test
6OPSVận hành — ops, status, monitor, sync, security
0METAXuyên suốt — help, cook, init, bootstrap, status
💡
Nguyên lý cốt lõi: PEV (Plan→Execute→Verify) · MCU billing · Water Protocol 水 · Jidoka (tự kiểm)

⚙ Cài Đặt & Cấu Hình

Thiết lập Mekong CLI trên máy của bạn.

# Cài đặt Python CLI
pip install -e .

# Cấu hình LLM — hỗ trợ MỌI nhà cung cấp
export LLM_BASE_URL="https://api.anthropic.com"
export LLM_API_KEY="sk-ant-..."
export LLM_MODEL="claude-sonnet-4-6"

# Kiểm tra sức khỏe
mekong doctor check

🔧 Tầng 0 — META

Các lệnh xuyên suốt — nền tảng cho mọi hoạt động.

/help

Hiển thị danh sách tất cả lệnh. Tự nhận diện persona (dev / non-tech) để hiển thị phù hợp.

mekong help              # Xem tất cả lệnh
mekong help cook         # Chi tiết lệnh cook
/cook⭐

Lệnh số 1 quan trọng nhất! Nhận mô tả bằng ngôn ngữ tự nhiên → tự phân loại → chọn agent + model → thực thi PEV pipeline → billing.

# CMO viết content marketing
mekong cook "viết email marketing cho bộ serum mới"

# CTO fix bug
mekong cook "sửa lỗi validation SĐT trong order-form" --agent cto

# Xem kế hoạch trước, không tốn MCU
mekong cook "thêm loyalty points" --dry-run

# Data agent phân tích
mekong cook "phân tích top 10 sản phẩm bán chạy Q1" --agent data
FlagMô tả
--agent <role>Chỉ định agent: cto · cmo · coo · cfo · cs · sales · editor · data
--model <id>Chỉ định model LLM cụ thể
--dry-runChỉ lên kế hoạch, không thực thi, không tính MCU
--no-billThực thi nhưng không tính MCU (dev/test)
--strictFail ngay nếu verify không pass
--verboseHiện chi tiết routing và quyết định
SCAN→ CLASSIFY→ MODEL SELECT→ PEV EXECUTE→ BILLING→ MEMORY
/init

Tự động setup toàn bộ cấu trúc AgencyOS. Không hỏi gì — zero questions, auto-detect.

mekong init              # Khởi tạo trong thư mục hiện tại
/bootstrap⭐

Orchestrator lớn nhất — toàn bộ quy trình: hỏi yêu cầu → research → tech stack → plan → design → code → test → review → deploy → docs.

mekong bootstrap "website đặt hàng mỹ phẩm Hàn, hỗ trợ COD, tích hợp Telegram"
🌸
Đây chính là cách dự án K-Beauty Order được khởi tạo!
/status

Dashboard sức khỏe: Ollama, API keys, MCU balance, git status, company config.

mekong status

🎨 Tầng 1 — STUDIO

Tầng sáng tạo & thiết kế — nơi ý tưởng được sinh ra.

/brainstorm

Động não ý tưởng. Nhận phân tích đa chiều với 2-3 giải pháp kèm pros/cons. Chỉ tư vấn, không code.

mekong brainstorm "nên dùng Shopify hay tự build cho K-Beauty?"
mekong brainstorm "chiến lược pricing cho dòng serum cao cấp"
/design

Tạo design guidelines, wireframes HTML cho developers.

mekong design "form đặt hàng mobile-first cho K-Beauty"
/branding · /content · /video

Brand identity (logo, colors) · Viết nội dung theo brand voice · Kịch bản video marketing.

mekong branding "K-Beauty VN — mỹ phẩm Hàn tại Việt Nam"
mekong content "mô tả COSRX Snail Mucin cho website"
mekong video "script TikTok 60s review skincare Hàn Quốc"

🏛 Tầng 2 — FOUNDER

Chiến lược cấp cao — tầm nhìn, huy động vốn, tình báo thị trường.

/intel

Thu thập tình báo: phân tích đối thủ, xu hướng thị trường, profile nhà đầu tư.

mekong intel "Hasaki.vn"                     # Deep dive đối thủ
mekong intel market "mỹ phẩm Hàn Quốc VN"   # Phân tích thị trường
🌸
Dùng mekong intel "Hasaki.vn" trước khi ra mắt để hiểu pricing, strengths của đối thủ.
/binh-phap · /swot · /okr · /fundraise · /competitor

Chiến lược Binh Pháp Tôn Tử · SWOT · OKR · Huy động vốn · So sánh đối thủ.

mekong binh-phap "chiến lược thâm nhập thị trường mỹ phẩm Hàn tại VN"
mekong swot "K-Beauty Order"
mekong okr "Q2 2026 — tăng trưởng đơn hàng 200%"
mekong fundraise "seed round $200K cho K-Beauty VN"
mekong competitor "so sánh K-Beauty với Hasaki, Lixibox"

💼 Tầng 3 — BUSINESS

Sales, marketing, tài chính, khách hàng.

/sales

Pipeline bán hàng: lead → qualify → proposal → close → invoice.

mekong sales                    # Pipeline overview
mekong sales forecast           # Dự báo doanh thu
Sub-commandCông dụng
/sales/leadTìm & qualify leads — ví dụ: tìm KOL mỹ phẩm
/sales/proposalTạo proposals — đề xuất hợp tác brand Hàn
/sales/closeClosing scripts — chốt deal nhà phân phối
/marketing

Hub marketing: content, ads, SEO, analytics.

mekong marketing                 # Dashboard
mekong marketing ideas           # Generate ý tưởng
mekong marketing calendar        # Lịch nội dung
Sub-commandVí dụ K-Beauty
/marketing/socialBài TikTok, Instagram về skincare routine
/marketing/emailEmail flash sale mỹ phẩm
/marketing/adsFacebook Ads cho serum mới
/marketing/seoTối ưu từ khóa "mỹ phẩm Hàn Quốc"
/finance

Quản lý tài chính: invoicing, expense tracking, runway, tax.

mekong finance                   # Dashboard
mekong finance forecast          # Dự báo 3-6 tháng
mekong finance report monthly    # Báo cáo tháng
/crm · /affiliate · /pricing · /leadgen

CRM khách hàng · Tiếp thị liên kết · Chiến lược giá · Tạo leads.

mekong pricing "định giá bộ skincare combo 3 bước"
mekong leadgen "tìm khách tiềm năng mỹ phẩm Hàn ở TPHCM"

📦 Tầng 4 — PRODUCT

Lập kế hoạch, quản lý sprint, phát hành sản phẩm.

/plan⭐

Chia mục tiêu lớn → subtasks nhỏ, tự phân công agent + model + tính MCU.

mekong plan "xây dựng hệ thống loyalty points cho K-Beauty"
mekong plan "tích hợp thanh toán MoMo và VNPay" --execute
/sprint · /roadmap

Quản lý sprint 1-2 tuần · Lộ trình sản phẩm dài hạn.

mekong sprint plan "2 weeks"     # Sprint mới
mekong roadmap "K-Beauty 2026"   # Product roadmap
/launch⭐

Tạo bản copy ra mắt cho Product Hunt, Hacker News, Reddit — kèm launch checklist.

mekong launch "K-Beauty Order" --tagline "Korean beauty, Vietnamese price"
/ship

Deploy feature ra production — pre-deploy checklist + rollback plan.

mekong ship feature "loyalty"    # Ship tính năng
mekong ship hotfix "bug123"      # Emergency fix

🛠 Tầng 5 — ENGINEERING

Viết code, debug, review, deploy — trái tim kỹ thuật.

/code⭐

Code theo implementation plan. 6 bước: Analysis → Implementation → Testing → Code Review → User Approval → Finalize.

mekong code                # Tự tìm plan mới nhất
mekong code "phase-01"     # Code phase cụ thể
/fix⭐

Debug BẮT BUỘC qua: SCAN → Root Cause → Jidoka Gate → Minimal Fix.

mekong fix "đơn hàng không hiển thị trên admin.html"
mekong fix "lỗi Telegram khi có emoji" --file tools/telegram_bot.py
⚠️
Jidoka: Tự động DỪNG khi đụng DB schema, payment, authentication. Cần CEO approve.
/review

Review theo domain: code (security, performance) · content (SEO, tone) · ops (idempotent).

mekong review order-form.html --full
mekong review telegram_bot.py --security
/deploy⭐

Multi-strategy: Blue-Green · Canary · Rolling · Recreate. Kèm CI/CD.

mekong deploy blue-green production   # Zero downtime
mekong deploy canary staging          # 10% traffic test
/test · /refactor · /debug · /lint · /schema

Test suite · Tái cấu trúc code · Debug sâu · Lint · Schema management.

mekong test e2e
mekong refactor "tách logic thanh toán ra module riêng"

⚙ Tầng 6 — OPS

Monitoring, billing, automation, đồng bộ hệ thống.

/ops

Hub vận hành: billing, reporting, team, automation.

mekong ops                 # Dashboard
mekong ops automate        # Gợi ý automation
mekong ops report weekly   # Báo cáo tuần
/sync-* · /monitor · /rollback · /security

Đồng bộ agents/tasks/artifacts · Giám sát metrics · Rollback khẩn cấp · Audit bảo mật.

mekong sync-all            # Đồng bộ tất cả
mekong monitor             # Dashboard metrics
mekong rollback "v1.1.0"   # Quay lại version trước
mekong security            # Audit bảo mật

🌊 Water Protocol 水

Cơ chế truyền tải ngữ cảnh đa agent giữa các tầng.

🎨 Studio──水──▶ 🏛 Founder──水──▶ 💼 Business──水──▶ 📦 Product──水──▶ 🛠 Engineering──水──▶ ⚙ Ops
🌸
K-Beauty flow: brainstorm (quyết định loyalty) → plan (4 subtasks) → code (CTO triển khai) → review → ship → ops monitor. Mỗi bước tự động truyền context qua .mekong/memory.json
